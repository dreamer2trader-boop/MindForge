# NeonFlow - PWA Installation Guide

NeonFlow is now a Progressive Web App (PWA) that can be installed on your device for a native app-like experience!

## Features
- **Offline Access**: Use the app even without internet connection
- **Quick Launch**: Access from your home screen like a native app
- **No App Store**: Install directly from your browser
- **Auto Updates**: Always get the latest version automatically
- **Full Screen**: Runs in standalone mode without browser UI

## Technical Details

- **Service Worker**: Caches app resources for offline access
- **Manifest**: Defines app appearance and behavior
- **Icons**: Adaptive icons for all platforms (120px - 512px)
- **Theme Color**: #00d4ff (Cyan)
- **Display Mode**: Standalone (fullscreen experience)

## Troubleshooting

**Install option not showing?**
- Make sure you're using HTTPS
- Try refreshing the page
- Clear browser cache and reload
- Check if you're using a supported browser

**Already installed but not working?**
- Uninstall and reinstall the app
- Clear app cache in browser settings
- Make sure you have the latest browser version
